# Macao LRT Web App (Testing)

A React + Vite web application for Macao Light Rapid Transit (LRT) information, including station details, route planning, and fare lookup.

## Features

- Station list and station detail pages
- Route planning between stations
- Fare query with ticket-type options
- Bilingual UI (Traditional Chinese / English)
- Timetable and translation data integration
- Station image assets for richer station pages
